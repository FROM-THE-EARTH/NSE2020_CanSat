# canbeta

東北大学 [FTE](https://www.fte-tohoku.org/) CanSat チーム9期代のプロジェクトです．

## Dependancies

[Pipfile](./Pipfile) を参照してください．
フレームワークは [pisat](https://github.com/jjj999/pisat) を使っています．

## 開発者へ
